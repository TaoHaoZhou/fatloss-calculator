# 减脂热量追踪计算器

一个完全离线的减脂期热量 & 宏量营养素追踪工具，浏览器打开即可使用，所有数据存储在本地。

## 功能特点

### 🔍 食物搜索
- 实时模糊搜索，支持食物名称和关键词
- **60+ 中式炒菜数据库**（宫保鸡丁、鱼香肉丝、回锅肉、麻婆豆腐、红烧肉 等）
- 快餐/外卖热量（麦当劳、黄焖鸡、麻辣烫、兰州拉面 等）
- 分类筛选标签（全部/主食/肉类/蔬菜/水果/奶类/坚果/炒菜/快餐/其他）

### 📊 热量与宏量追踪
- BMR（Mifflin-St Jeor 公式）
- 性别、运动强度自定义
- 可调热量缺口（默认 500kcal）
- 碳水 / 蛋白 / 脂肪分别追踪，带进度条

### ⚖️ 体重追踪
- 空腹体重每日记录，与上次对比（↑↓→ 直观显示）

### 📈 数据可视化
- 近 30 天趋势图表（体重 / 热量 / 蛋白三视图）
- 近 7 天周报

### 💾 数据管理
- JSON 数据导出 / 导入
- CSV 历史记录导出
- 完全离线，无服务器依赖

## 使用方法

直接在浏览器打开 `fatloss-calculator.html` 即可，无需安装任何软件。

## 数据来源

食物热量数据参考中国食物成分表、USDA 食品数据库及薄荷健康等公开数据。
